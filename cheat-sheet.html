Common CSS Properties

CSS

CSS is a language used along side HTML and is used to style your websites. Where HTML creates the structure of the webpage, CSS Creates a better stylistic appearance for it. CSS can be used within your HTML file either inline or with the style element. To keep things neater and more readible I would suggest using a seperate CSS stylesheet and link them together. To do so use the link element with attributes href, type and rel. Href is used to attach the .style.css file. Type is used to specify it is written in text/css and rel is used to specify it is linking to a related stylesheet.

CSS Selectors and Specificity

Before going on to view the common CSS properties you should first be familiar with CSS selectors and their specificity. When writing CSS you should always use the least specific selector for the part of HTML you are using. This is so later if you choose to add another CSS property you can use a more specific tag and this will override your previous code. So fthe least specific selector is the tag e.g h1 or p. Then the class e.g in HTML this would be added to a element with class="nameofclass" and on css appears as .nameofclass. Finally the id e.g. on HTML would appear as atrribute id="nameofID" on CSS as #nameofid. To over ride all of these- which should be used only if absolutely necessary- use the !important attribute after the css declaration.
